# Example Package

This is a simple example package. You can use
[GitHub-flavored Markdown](https://guides.github.com/features/mastering-markdown/)
to write your content.

install in editable mode
```cmd
pip install -e .
```

running modules
```cmd
py -m databricks.file_name
```

build module
```cmd
py -m build
```

install module with optional reinstall [--force-reinstall]
```cmd
pip install "libraries/dist/databricks-0.0.1-py3-none-any.whl"
```
